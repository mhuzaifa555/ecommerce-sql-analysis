# E-Commerce Sales Analysis (SQL Project)

This project analyzes a simple e-commerce database using SQL.

## Database Tables

- Customers
- Products
- Orders
- Order Items

## Objectives

The goal of this project is to analyze sales performance and customer activity using SQL queries.

## Key Business Questions

1. Which products sell the most?
2. Which products generate the highest revenue?
3. Which customers place the most orders?
4. What is the revenue from delivered orders?

## Skills Used

- SQL Joins
- GROUP BY
- Aggregations (SUM, COUNT)
- Data analysis using relational databases

## Tools

- MySQL
